Responsive Design

Responsive Design allows pages to work well on desktop, tablet, and mobile devices, regardless of screen size.

        **WITHIN THE CSS FILE**

        @media (min-width: 992px)
            .col-md-3 {
          width: 25%;
        }

        **DESCRIPTION**

        The @media in the external stylesheet is a media query which allows users to apply certain styles when the screen hits a certain
        point. Here in the example we see that when the media display is 992px or over, it will display the medium column of three with a
        width of 25%.
