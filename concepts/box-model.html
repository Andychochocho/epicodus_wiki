Box Model

The box model has elements on a web page takes up space, and the box model describes the space around the element.

Here's an example:

        
**WITHIN THE CSS FILE**

        
.my-class {

        
  margin: 12 px;

        
  border: 2px solid green;

        
  padding: 4px;

        
  background-color: orange;

        
}
