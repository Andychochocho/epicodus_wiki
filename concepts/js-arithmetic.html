Arithmetic Operators

Arithmetic operators take numerical values (either literals or variables) as their operands and return a single numerical value. The standard arithmetic operators are addition (+), subtraction (-), multiplication (*), division (/), and remainder (%).

        **WITHIN THE JAVASCRIPT CONSOLE**
        
        > 1 + 2
        3

        > 15 - 7
        8

        > 5 * 0
        0

        > 20 / 4
        5

        > 5 % 4
        1

        **DESCRIPTION**

        In the example given above, we see that you can add or subtract different intergers from each other. Further down you can also see
        that you can multiply intergers with one another. Finally we have a remainder operator that we can see gives the remainder when we
        divide five divided by four.
